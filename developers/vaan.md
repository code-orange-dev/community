## Why This Work Matters

BDK (Bitcoin Dev Kit) is the Rust wallet library underpinning dozens of production Bitcoin applications. rust-bitcoin provides the core Bitcoin primitives — serialisation, transaction types, script, keys — used by virtually every Rust Bitcoin project. Contributions to these libraries have multiplicative impact: improvements flow through to every application built on top of them.
