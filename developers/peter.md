**Active contributions:**
- **Bitcoin Core** — C++ contributions to the reference implementation (some under review)
- **rust-bitcoin** — contributing to the foundational Rust Bitcoin library
- **Kernel-Node** — work on the Bitcoin kernel/node abstraction layer (some under review)

## Why This Matters

Bitcoin Core contributors are rare globally — fewer than 200 developers have ever made meaningful contributions. Peter's simultaneous work in Core (C++) and rust-bitcoin (Rust) shows breadth across Bitcoin's two dominant technical stacks. The Kernel-Node work is particularly significant — it relates to the ongoing effort to modularize Bitcoin Core into a reusable kernel library.
