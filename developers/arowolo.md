## About

Arowolo recently had their first pull request approved on [rust-payjoin](https://github.com/payjoin/rust-payjoin) — the Rust implementation of Async Payjoin (BIP77). This is a significant milestone: rust-payjoin is a critical privacy project with 37 contributors, maintained by Dan Gould (Payjoin Foundation executive director).

Payjoin is not a peripheral tool. If even 5-10% of Bitcoin users adopt it, the common input ownership heuristic — the primary surveillance mechanism used by blockchain analysis firms — becomes unreliable for all Bitcoin users, not just Payjoin users.
