## Why This Work Matters

peer-observer is a Bitcoin network security tool created by 0xB10C (an OpenSats long-term support grantee) that monitors Bitcoin's peer-to-peer network for attacks and anomalies using Bitcoin Core honeynodes. It is critical security research infrastructure for the Bitcoin network. Razor's contributions improve the tooling that helps detect and understand threats to Bitcoin's P2P layer.
