## What They're Building

Psychemist is building directly at the Bitcoin application and library layer with a clear trajectory toward protocol-level contribution.

**Current work:**
- **BDK QR WIF sweep on Android** — building a Kotlin/Android tool to sweep wallet keys via QR + WIF format using Bitcoin Dev Kit
- **BIP375 in Go** — implementing the Silent Payment Output Descriptors spec in Go
- **rust-ldk contribution** — contributing to LDK (Lightning Dev Kit), the Rust Lightning implementation used in production wallets

**Next target:** Bitcoin Core FFI/RPC interface issues — the bridge between Core and higher-level applications.

---

## Why This Matters

Psychemist is one of the few Code Orange contributors working across multiple Bitcoin layers simultaneously — Android wallet tooling, Go protocol implementations, and Rust Lightning. The BIP375 Go implementation is particularly notable: it fills a gap in non-Rust BIP352/375 tooling that the broader ecosystem needs.
