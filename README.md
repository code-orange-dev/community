# Code Orange Dev School — Community Contributors

> **We build the community. They build Bitcoin.**

This repository documents the developers who are part of the Code Orange Dev School community. These are independent contributors — Code Orange provides mentorship, structured cohorts, and a global peer network. Their contributions belong to them. We are proud to support them.

**Our goal: 2 pull requests per week to Bitcoin open-source software from this community.**

---

## Why This Community Matters

Bitcoin Core has approximately 135 active contributors globally. The vast majority come from North America and Western Europe. The developers listed here come from Indonesia, India, Zambia, Korea, Germany, Romania, and beyond — regions deeply underrepresented in Bitcoin development, and often the exact regions where Bitcoin matters most as a tool for financial freedom.

Code Orange's education-to-contribution pipeline is designed to change that distribution — one merged PR at a time.

## Our 2 PR / Week Target

**Goal:** 2 pull requests per week from this community to Bitcoin open-source software.

**Current baseline (verified):**
- 7 active contributors currently producing PRs (Chaitika, Vaan, Razor, Arowolo, Psychemist, Diegodev, Peter)
- 2 more with first PRs imminent (Gradale, Muhammad)
- Conservative projection: ~150 PRs/year ≈ 2.9 PRs/week without additional support
- With Code Orange mentorship funding: ~200+ PRs/year ≈ 3.8 PRs/week

This is not a projection built on hope. It is built on the documented momentum of developers who are already shipping code.
